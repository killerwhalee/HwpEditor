HML 클래스 개요

HML(%파일경로%)로 HML 오브젝트 생성

오브젝트의 메소드
getText() -> 문서 내 텍스트를 추출합니다
getTextList() -> 문서의 텍스트를 리스트로 추출합니다

오브젝트의 인스턴스
textStrList -> 문서의 텍스트를 Text 오브젝트로 구성된 리스트로 저장합니다
hmlTree -> 파싱된 문서의 모든 데이터를 저장합니다. 사람이 읽기에는 복잡합니다.

----------------------------------------

Text 클래스 개요

Text 오브젝트의 인스턴스
type -> Text의 타입입니다. CHAR는 일반 문자열, EQUATION은 수식, LINEBREAK는 빈 줄바꿈 문자열입니다.
style -> Text의 글자 모양입니다. 글꼴이나 자간, 장평 등의 정보를 담습니다. 아직 구현되지 않았습니다. 빈 딕셔너리를 출력합니다.
string -> Text가 저장한 문자열입니다. EQUATION의 경우 수식 스크립트가 저장됩니다.
LINEBREAK는 \n을 저장합니다.
